# 🎮 Pasaporte de Pasante

Tu repo personal de la pasantía. Acá vas guardando cada misión que completás, tus badges y tu XP. Al final de los 3 meses, esto es tu **portfolio** para mostrar.

## Reglas del juego

1. **Una misión = un branch + un PR**. Nunca commitees a `main` directo.
2. **Conventional Commits siempre**. Ver [docs/conventional-commits.md](docs/conventional-commits.md).
3. **Podés usar AI** (v0, Claude, Cursor) pero en el PR tenés que explicar *qué hace cada parte* del código. Sin entender = no merge.
4. **Code review obligatorio**. El game master (Roman) revisa. Si pasa, mergeás y ganás el badge.
5. **Si no entendés algo, preguntá**. No hay penalización por preguntar. Sí hay penalización por copiar sin entender.
